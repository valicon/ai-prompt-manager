# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),

## [1.0.0] - 2025-03-13

### Added

- Initial release
- `prompt_upgrade` MCP tool that calls PromptLab `/upgrade` endpoint
- Configurable `PROMPTLAB_URL` environment variable (default: `http://localhost:3000`)
- Runnable via `npx -y promptlab-mcp` from any project
